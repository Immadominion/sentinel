features:
  - icon: 🔑
    title: セッションキー
    details: エージェントは時間と金額が制限された一時的なキーを受け取ります。ウォレットの秘密鍵を持つことはありません。セッションは自動的に期限切れになります。

  - icon: 💰
    title: 日次支出制限
    details: 各エージェントが1日および1トランザクションで使用できる金額を設定します。オンチェーンプログラムが適用します——例外なし。

  - icon: 🛡️
    title: プログラム許可リスト
    details: LPボットはMeteoraに留まります。スナイパーボットはRaydiumに留まります。侵害された場合でも、エージェントは承認されていないコントラクトを呼び出せません。

  - icon: 🚨
    title: 緊急停止スイッチ
    details: 1回の呼び出しで全エージェントを即座にフリーズします。常に完全な制御を維持できます。

  - icon: 👥
    title: ガーディアン回復
    details: キーを紛失しましたか？ガーディアンがマルチシグでウォレット権限をローテーションできます。資産は常に回復可能です。

  - icon: 🤖
    title: マルチエージェントサポート
    details: ウォレットごとに最大5つの完全独立したエージェント。LPボット、スナイパー、イールド最適化——それぞれ独自の制限を持ちます。
